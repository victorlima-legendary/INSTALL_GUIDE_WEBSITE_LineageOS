1 - What do you need to continue?

  • Your device unlocked, if you don't unlock yet, follow this tutorial;  click here
  • Platform Tools downloaded in your computer;  click here
  • Motorola USB drivers installed in your computer;  click here

If you using Linux you need to install fastboot packages:

install adb packages for linux users (if you are on windows, skip this):
sudo apt update && sudo apt install adb fastboot
Connect your device in fastboot mode, and try this command:
fastboot devices

2 - STEP-BY-STEP TO INSTALL LINEAGEOS ON MOTO G82 5G:

  • DOWNLOAD PLATFORM TOOLS AND EXTRACT, LATER OPEN A TERMINAL INSIDE THIS FOLDER.
  • FIRST YOU NEED TO INSTALL ADB AOSP RECOVERY;  click here
STEP-BY-STEP TO INSTALL ADB RECOVERY FILES ON THE DEVICE:
fastboot flash vendor_boot vendor_boot.img && fastboot flash boot boot.img && fastboot flash dtbo dtbo.img

Later reboot device into recovery mode.

INSTALL DOWNLOADED CUSTOM ROM FILE:

- All image here is from Vishuu
  • In the recovery, select factory reset and later format data.
  • Return to inicial recovery screen and then select Apply Update and Apply from ADB
  • Now, in the computer, move the ROM file into platform and execute this command in the command prompt:

    adb sideload  "Custom ROM directory"
  • Once the installation is complete, click no and return to the home recovery screen.
  • Now, click on factory reset and format data
  • Now you can boot the ROM.  Enjoy!!

3 - HOW CAN WE INSTALL THE GAPPS?

Step-By-Step to install the Gapps:
  • First you need to download the Gapps of your preference... My sugestion;  click here!

  • Now, Reboot your device into recovery mode, and click on Apply Update and Apply from ADB



  • Now execute the command bellow into the command prompt:

    • NOTE:  you need to open the command prompt inside the platform tools directory!!

    adb sideload  "< Gapps directory >"

    NOTE:  If you are going install the omini gapps or stock gapps, you need to do factory reset first!!

  • After the gapps installation is complete, you can do factory reset and reboot your device.

    • Enjoy Your Device!!
